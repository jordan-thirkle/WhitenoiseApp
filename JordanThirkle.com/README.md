# 🚀 Jordan Thirkle | Creator Hub

A highly optimized, minimalist developer portfolio and blog built with **Astro 4+**, **React Islands**, and **Tailwind CSS**.

## 🛠️ Tech Stack
- **Framework**: [Astro](https://astro.build/)
- **UI Architecture**: React Islands (`client:load`, `client:idle`)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Zinc palette)
- **State Management**: [Nanostores](https://github.com/nanostores/nanostores)
- **Content**: MDX + Astro Content Collections
- **Animations**: Framer Motion
- **Icons**: Lucide React + Astro Icon

## 🏃 Local Development

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd JordanThirkle.com
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open the site:**
   Navigate to `http://localhost:4321`.

## 📁 Key Directories
- `src/content/`: Your blog posts and case studies.
- `src/components/react/`: Interactive islands.
- `src/components/astro/`: Static UI components.
- `src/layouts/`: Page scaffolds.

## 🚀 Deployment
This project is pre-configured for **Vercel** or **Netlify**. Simply connect your GitHub repository and it will build and deploy automatically.
